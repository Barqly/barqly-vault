Why Donate?

Barqly Vault is developed as an open-source project to serve the Bitcoin community and anyone who needs secure file encryption. Your donations help cover development costs, security audits, and infrastructure needed to make this tool available to everyone.

Make a Donation

Choose your preferred amount and currency. All donations go directly toward development and maintenance.

What Your Donation Supports

  • Development: Core features, bug fixes, and platform support
  • Security: Regular security audits and cryptographic reviews
  • Infrastructure: Build servers, code signing certificates, and hosting
  • Documentation: User guides, tutorials, and developer resources
  • Community: Support channels and user assistance
